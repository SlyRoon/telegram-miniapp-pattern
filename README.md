# scalable-frontend-pattern

Minimal Core starter on React + TypeScript + Vite.

## Included Core

- React Router (Home + NotFound routes)
- Redux Toolkit store with typed hooks
- i18next + react-i18next (en/uk)
- Global react-hot-toast provider
- Axios API client (`src/shared/api/client.ts`)
- Tailwind + shadcn init + `@` alias + SVGR

## Scripts

- `npm run dev`
- `npm run lint`
- `npm run build`
- `npm run preview`

## Project Structure

- `src/app` - app composition, providers, router, store
- `src/pages` - route pages
- `src/shared` - shared api/i18n/lib
